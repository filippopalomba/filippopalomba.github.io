scpi: software package for prediction and inference using synthetic controls, implemented in Python, R, and Stata. For point estimation or prediction of treatment effects, the package offers an array of (possibly penalized) approaches leveraging the latest optimization methods. For uncertainty quantification, the package offers the prediction interval methods introduced by Cattaneo, Feng, Palomba, and Titiunik (2022).
CRAN | PyPI

getaway: software package that implements point estimation and inference away from the cutoff in Regression Discontinuity (RD) designs as proposed in Angrist and Rokkanen (2015).
